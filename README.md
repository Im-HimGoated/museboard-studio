# Museboard Studio

Museboard Studio is a static website prototype for a shared creative workspace. It is designed for designers and creative teams who need one place to capture scattered ideas, organize references, connect concepts, and track project momentum.

## Project Goal

Create one shared workspace where creative teams can turn scattered inspiration from emails, notes, screenshots, links, documents, and sketches into organized project progress.

## MVP Features

- Quick capture for text ideas, image notes, links, and sketches
- Project-based navigation for active creative work
- Flexible board layout for grouping and rearranging ideas
- Connected idea cards for notes, references, feedback, and next steps
- Basic task tracking with progress and completion states
- Search and filter controls represented in the interface

## Files

- `index.html` - Main website structure
- `styles.css` - Visual design and responsive layout
- `script.js` - Prototype interactions

## How to Run

Open `index.html` directly in a browser.

No build step or package installation is required.

## GitHub Pages

This project can be published with GitHub Pages:

1. Upload the files to a GitHub repository.
2. Go to repository `Settings`.
3. Open `Pages`.
4. Set the source branch to `main`.
5. Save and wait for GitHub to generate the public URL.

## Prototype Status

This is a front-end prototype. The interface demonstrates the product concept and core interactions, but it does not yet include authentication, persistent storage, real file uploads, or integrations with Google Docs, Slides, Sheets, or email.
